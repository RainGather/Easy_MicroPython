约定
====================
中英文
====================

为了方便中文使用者，本项目中多种函数、类都有对应的中文拼音名可用，两者效果是一致的。

通用模板
====================
以下为通用模板::

    from ezmpy import *
    
    wifi('WIFI名称', 'WIFI密码')  # 可选，连接wifi，默认会同时连接test.mosquitto.org用作MQTT服务器

    # 在这里写你的代码

    run()  # 开始执行循环守护进程

如果没有from ezmpy import *或run(), 系统在烧录时会自动加上。
