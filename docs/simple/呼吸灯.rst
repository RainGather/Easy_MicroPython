呼吸灯
=============

将LED正极连接到针脚2，用如下代码可以控制LED灯变成呼吸灯::

    # 导入时间库，可以使用等待功能
    import time
    from ezmpy import *


    # 声明2针脚为PWM输出端，在ESP8266中针脚0, 2, 4, 5, 12, 13, 14 和 15支持PWM模拟输出 
    p = PWM(2)


    while True:
        # 将指针2的输出，从0逐渐增长到1024，0为最低输出（0V），1024为最高输出（5V或3.3V）
        for i in range(1024):
            # 设置针脚输出
            p.duty(i)
            # 等待0.005秒，防止闪烁过快
            time.sleep(0.005)
        # 将指针2的输出，从1024逐渐减少到0
        for i in range(1024, 0, -1):
            # 设置针脚输出
            p.duty(i)
            # 等待0.005秒，防止闪烁过快
            time.sleep(0.005)
