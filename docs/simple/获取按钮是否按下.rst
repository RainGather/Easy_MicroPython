获取按钮是否按下
==================

将按钮分别将针脚0连接到GND和VCC，并用如下代码获取针脚0连接的是GND还是VCC::

    # 导入时间库，这样可以使用等待的功能
    import time


    # 将针脚0作为数字输入
    p = IN(0)


    while True:
        # 获取针脚0上输入的电平高低，如果为高则为1，如果为低则为0
        i = p.value()
        # 将其值打印出来
        print(i)
        # 等待1秒，防止刷新太快
        time.sleep(1)
