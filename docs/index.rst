Easy MicroPython 教程与手册
============================================

Easy MicroPython是在MicroPython之上的一层封装，将针脚操作和传感器、控制器连接进行了进一步的简化，目的是为了让初学者更加方便、快捷地使用MicroPython。
本项目目前仅在基于ESP8266或ESP32的NodeMcu上进行过调试，其余硬件不保证兼容性。
